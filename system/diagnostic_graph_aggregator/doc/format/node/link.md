# Link

The `link` object is a node that refers to other nodes.

## Format

| Name   | Type     | Required | Description                            |
| ------ | -------- | -------- | -------------------------------------- |
| `type` | `string` | yes      | Specify `link` when using this object. |
| `link` | `string` | yes      | The path of the node to reference.     |
